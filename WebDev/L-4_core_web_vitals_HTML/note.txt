First Contentful Paint (FCP) is a web performance metric that measures the time it takes for the browser to render the first piece of content from the DOM (Document Object Model) on the screen.



Largest Contentful Paint (LCP) is a user-centric web performance metric that measures the loading time of the largest and most meaningful content element in the viewport.

 


Cumulative Layout Shift (CLS) is a user-centric web performance metric that measures the visual stability of a web page. It quantifies how much the elements on a page move or shift unexpectedly during the page's loading process. CLS is crucial for assessing and improving the user experience, as layout shifts can lead to user frustration, accidental clicks, or content being pushed out of view.
